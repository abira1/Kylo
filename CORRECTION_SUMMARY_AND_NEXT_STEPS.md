## 🤖 THE AI SYSTEM (Core Product - Was Missing)

**This is the entire product intelligence.** Every response comes from Claude API.

### Architecture:
```
User Message → Q&A Retrieval → Claude Prompt Builder → 
Claude API Call → Response Generation → Lead Capture Logic →
Firestore Storage → Frontend Display
```

### Key Components:

**1. Q&A Knowledge Base Retrieval**
- Extract keywords from user message
- Query Firestore for matching entries
- Global Q&A + client-specific Q&A
- Return top 6-8 results by usage count

**2. System Prompt Template**
- Client branding injection
- Role/identity of agent
- Tone and behavior instructions
- Q&A context injection
- Conversation history (last 8 messages)
- Current user message

**3. Claude API Configuration**
- Model: claude-sonnet-4-6
- Max tokens: 800 (concise responses)
- Temperature: 0.7 (natural, not hallucinating)
- Prompt caching: ENABLED (~90% token cost reduction)

**4. Lead Capture Logic**
- Turn counter trigger (after 3 responses)
- Intent keyword trigger (ready, apply, start, etc.)
- Automatic inline card in chat (not modal)

**5. Multi-channel Support**
- Web chat widget
- WhatsApp Business integration
- Unified conversation storage
